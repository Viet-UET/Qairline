# QAirline Frontend

This is the frontend for the QAirline booking system, built with React and Vite.

## Prerequisites

- Node.js (v18 or higher recommended)
- npm

## Getting Started

1.  **Install Dependencies**

    ```bash
    npm install
    ```

2.  **Run the Development Server**

    ```bash
    npm run dev
    ```

    The application will be available at `http://localhost:3000`.
